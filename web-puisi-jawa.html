4. Apakah perbedaan antara web browser,web page,web server


Web browser, web page, dan web server adalah tiga istilah yang terkait dengan penjelajahan halaman di dunia maya yang dilakukan oleh pengguna internet.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menyajikan perbedaan antara web browser, web page, dan web server. Berikut kakak akan mencoba menjawab pertanyaan tersebut.

WEB BROWSER ADALAH PERANGKAT LUNAK YANG DIPAKAI PENGGUNA INTERNET UNTUK BERSELANCAR DI DUNIA MAYA.

WEB PAGE ADALAH HALAMAN DI INTERNET YANG MENYAJIKAN INFORMASI KEPADA PENGGUNA DAN HANYA DAPAT DIBUKA DENGAN WEB BROWSER.

WEB SERVER ADALAH SISTEM ATAU PERANGKAT YANG DIGUNAKAN PIHAK PENYEDIA KONTEN UNTUK MENYAJIKAN ATAU MENAMPILKAN WEB PAGE DI INTERNET SERTA MENGAKOMODIR AKTIVITAS PENGGUNA DI WEB PAGE TERSEBUT.

9. Web atau situs web memiliki halaman web yang berisi data dan informasi yang tersimpan di web server dalam bentuk​


﻿﻿Web atau situs web biasanya terdiri dari halaman-halaman web yang berisi data dan informasi dalam bentuk teks, gambar, video, dan elemen-elemen multimedia lainnya. Halaman-halaman web ini disimpan di server web dalam bentuk file HTML, CSS, JavaScript, dan file media lainnya.

﻿File HTML (Hypertext Markup Language) adalah dasar dari semua halaman web dan digunakan untuk membangun struktur halaman web. File CSS (Cascading Style Sheets) digunakan untuk memperindah tampilan halaman web dengan mengatur warna, font, dan layout. Sedangkan file JavaScript digunakan untuk membuat halaman web interaktif dan dinamis.

﻿Selain itu, halaman web juga dapat menyimpan data dan informasi dalam bentuk file media seperti gambar, video, dan audio. File-file media ini disimpan di server web dan dapat diakses oleh pengguna melalui halaman web.

﻿Secara umum, semua data dan informasi yang terdapat di halaman web disimpan di server web dalam bentuk file yang kemudian diakses oleh pengguna melalui browser web.

Aksara Jawa

Jawaban Terlampir

PEMBAHASAN

Aksara Jawa yaiku tulisan jawa utawa aksara tradisional sing berkembang nang Pulau Jawa.

Aksara Jawa cacahe ana 20 :

•Ha, Na, Ca, Ra, Ka

•Da, Ta, Sa, Wa, La

• Pa, Dha, Ja, Ya, Nya

• Ma, Ga, Ba, Tha, Nga

Sandhangan Aksara Jawa ana 3 :

1. Sandhangan Swara

2. Sandhangan Panyigeg Wanda

3. Sandhangan Wyanjana

✺Semoga Membantu✺

Nulis aksara Jawa

Jawaban terlampir

Pembahasan

Nulis aksara Jawa iku sing kudu dimangerteni yaiku pasangan lan sandhangan.

1. Kumis pak Raden

kumis, aksara Ka + suku, aksara Ma + wulu, aksara Sapak, pasangan aksara Pa, aksara KaRaden, pasangan yang Ra, aksara Da + taling, aksara Na + pangkon.

2. Pis@nge dakgoreng

pis@nge, Aksara Pa + wulu, aksara Sa, aksara Nga + talingdakgoreng, Aksara Da, Aksara Ka, pasangan aksara Ga + taling tarung, aksara Ra + taling + cecak.

3. Pulpen iki werna biru

pulpen, aksara Pa + suku, aksara La, pasangan aksara Pa + taling, aksara NaIki, pasangan yang Ha + wulu, aksara Ka + wuluwerna, aksara Wa + pepet + layar, aksara Nabiru, Aksara Ba + wulu, aksara Ra + suku.

4. Bukumu tak ampil

bukumu, aksara Ba + suku, aksara Ka + suku, aksara Ma + sukutak, aksara Ta , aksara Kaampil, pasangan aksara Ha, aksara Ma, pasangan aksara Pa + wulu, aksara La + pangkon.

5. Baskom wadah roti

baskom, aksara Ba, aksara Sa, pasangan aksara Ka + taling tarung, aksara Mawadah, pasangan aksara Wa, aksara Da + wignyanroti, aksara Ra + taling tarung, aksara Ta + wulu

6. Dhik Lala nangis

aksara Dha + wulu, aksara KaLala, pasangan aksara La, aksara Lanangis, Aksara Na, aksara Nga + wulu, aksara Sa + pangkon.

7. Simbah sare

Simbah, aksara Sa + wulu, aksara Ma, pasangan aksara Ba + wignyansare, aksara Sa, aksara Ra + taling

8. ngontel sepeda

ngontel, aksara Nga + taling tarung, aksara Na, pasangan aksara Ta + taling, aksara Lasepeda, pasangan aksara Sa + pepet, aksara Pa + taling, aksara Da

9. Sik nyapu latar

sik, aksara Sa + wulu, aksara Kanyapu, pasangan aksara Nya, aksara Pa + sukulatar, aksara La, aksara Ta + layar

10. Mangan wajik

mangan, aksara Ma, aksara Nga, aksara Nawajik, pasangan aksara Wa, aksara Ja + wulu , aksara Ka + pangkon.

33. jelaskan tugas dari web desaigner,web programer,web administrator, web master?


Web desaigner : Mendesain tampilan dari pada website.
Web Programmer : Bertugas dalam melakukan pengcodingan atau pemograman suatu web agar dinamis.
Web administrator : Memaintenance satu buah server, mengerti akan Sistem Operasi Server, baik itu hal instalasi hingga kepada masalah-masalah pada web (troubleshooting)
Web master : Seorang yg sangat mengerti akan kesemua faktor mulai dari desain, program & keamanan server, meskipun demikian seorang webmaster tidak ikut mencampuri pada masing-masing divisi, cukup dgn tugas mempertanggung jawab atas jalannya suatu web (website).
