1. sebuah pamplet memiliki luas 16.1 m² jika lebar pamplet 3,5m berapa keliling pamplet?​


16,1m² ÷ 3,5m = 4,6m <= Panjang

(3,5m + 4,6m) × 2

= 8,1 × 2

= 16,2m <= Keliling

#SemogaMembantu

Penjelasan dengan langkah-langkah:

Panjang pamplet :

L = p·l

16,1 = p·3,5

16,1 = 3,5p

p = 16,1/3,5

p = 4,6 m

Keliling pamplet :

= 2·(p + l)

= 2·(4,6 + 3,5)

= 2·8,1

= 16,2 m
